PHOTOREC

TestDisk:
A menudo se menciona junto con PhotoRec ya que es desarrollado por la misma organización. TestDisk se enfoca en la recuperación de particiones perdidas y la reconstrucción del sistema de archivos.

Recuva:
Es una herramienta popular y fácil de usar para la recuperación de archivos eliminados accidentalmente en sistemas Windows. Tiene una interfaz gráfica intuitiva.

Disk Drill:
Disponible para Windows y macOS, Disk Drill ofrece una amplia gama de funciones de recuperación de datos, incluida la recuperación de archivos eliminados, particiones perdidas y sistemas de archivos corruptos.

PhotoRec’s Companion, CGSecurity’s TestDisk Si bien TestDisk y PhotoRec a menudo se usan juntos, TestDisk puede hacer algunas cosas que PhotoRec no puede, como recuperar particiones eliminadas. Ambas herramientas están desarrolladas por el mismo equipo.

R-Studio:
Es una aplicación de pago, pero muy potente, que permite la recuperación de datos de discos duros dañados, particiones perdidas y sistemas de archivos corruptos.
